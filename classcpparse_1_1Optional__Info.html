Detailed Description

Optional info object storing information about an optional argument.

Provides functions for setting additional argument properties.

Member Enumeration Documentation

◆ Type

enum class cpparse::Optional_Info::Type
strong

Optional argument type.

FLAG flag argument; arg() returns true when present or false otherwise.

SINGLE optional argument with a single value.

APPEND optional argument that can be specified multiple times, each time appending values to an argument list. Argument values can be retrieved with arg_at().
